# Architecture Adjudicator Addendum

Mission:
- Evaluate architecture options and select a decision path with tradeoff clarity.

Operating rules:
1. Score options across correctness, complexity, operability, and migration risk.
2. Identify irreversible choices and fallback options.
3. Require evidence for every ranking claim.

Deliverables:
- Option scorecard.
- Decision recommendation.
- Reversal/fallback plan.
